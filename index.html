귀농인들을 위한 웹사이트 - farm diary

2021.04 ~ 2021.11 (3인 팀 프로젝트)
농업인들에게 정보를 제공해주고, 물품 구매 판매, 날씨 정보를 확인 할 수 있는 웹 사이트입니다. 공공데이터 활용 경진대회에서 수상한 아이디어를 바탕으로 진행한 팀 프로젝트입니다.

OpenWeatherMap api, Kakao api, Youtube data api 등 여러 api를 사용하면서 사용법을 학습할 수 있었고, 도메인을 직접 구매해 AWS EC2 인스턴스 서버에 직접 호스팅 하면서 서버에 대한 학습도 할 수 있었던 프로젝트였습니다.
주요 기능
농업에 필요한 정보 검색, 지역별 습도, 강우량, 날씨 정보 확인, 물품 등록 구매 및 판매, 게시판 등록
